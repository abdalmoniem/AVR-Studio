Features:
=========
1- Debian based
2- Plain text editor.
3- Compiling .c files to various atmega parts or microcontrollers.
4- Uploadind .hex files to various atmega parts or microcontrollers.
5- Console area shows results of compilation and uploading.

Version:
========
Beta version

Installation:
==============
1- Download "AVR-Studio.zip" from website.
2- Open terminal.
3- cd to the directory where you saved the file.
4- issue the following commands:
	sudo ./install.sh
	exit
5- To start AVR-Studio simply type avr-studio in a terminal window or find it in the Dash
